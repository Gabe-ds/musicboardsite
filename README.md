# musicboard
課題で作成した音楽スレ

## 必要なもの
### 環境
- pyenv 
- pipenv
### 詳細
- python 3.9.6
- django 3.8.2
- psycopg2-binary
- django-bootstrap-v5
### settings_local.py
- settings.pyの拡張

## 導入
### python設定
```
pyenv install 3.9.6
pyenv local 3.9.6
pipenv --python 3.9.6
pipenv shell
```
### 必要ならライブラリ
```
pipenv install django psycopg2-binary django-bootstrap-v5
```
